AJAX Control Toolkit — November 2011 Release

This release of the Ajax Control Toolkit introduces a new BalloonPopupExtender control. This control enables you to display content which appears in a balloon popup when you move focus to a control, you click a control, or when you hover over a control. You can use the BalloonPopupExtender, for example, to display help messages when you fill out a form.

This release of the Ajax Control Toolkit also includes several highly requested features for the Tabs control

  • Vertical Tabs - You can display a vertical list of tabs on either the left or right-side of the Tabs control.
  • On-Demand Content Loading - You can use the new OnDemand property to control exactly when the contents of a tab panel is updated. For example, you can reload the contents of a tab panel whenever you select a tab, only the first time you select a tab, or never.
  • Keyboard Navigation - You can use the arrow keys, the Home key, or the End key to navigate from tab to tab.
  • Bug Fixes - This release of the Ajax Control Toolkit includes several important bug fixes for the Tabs control.
